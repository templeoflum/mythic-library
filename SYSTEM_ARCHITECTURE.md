## Validation Rules

### Layer 1: Calibration Integrity
- [x] Origin at exact center (0.5 all axes)
- [x] All poles at correct positions
- [x] Polar pairs have distance = 1.0 on their axis
- [x] Geodesics defined

### Layer 2: Primordial Coherence
- [x] All 22 primordials have unique coordinates
- [x] Shadow dynamics defined for each
- [x] Relationship matrix complete

### Layer 3: Domain Validity
- [x] All entries have complete 8-axis coordinates
- [x] All coordinates in [0.0, 1.0] range
- [x] All entries instantiate at least one primordial
- [x] No coordinate collisions (distance > 0.05)
- [x] Instantiation weights valid (0.0-1.0)

### Layer 4: Correspondence Consistency
- [x] Cross-system mappings include fidelity scores
- [x] Relationship types properly typed
