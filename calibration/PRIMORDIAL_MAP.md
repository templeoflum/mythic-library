### The Chaos Triad

- All cluster on chaos side

## Usage in v1.0

All 302 archetypes in v1.0 use this map:
- Each archetype `instantiates` one or more primordials with weights
- Spectral coordinates derived algorithmically from primordial positions
- Cross-system relationships (`CULTURAL_ECHO`) validated against primordial distances

### Example Instantiation
```json
{
  "name": "The Fool",
  "instantiates": [
    {"primordial": "primordial:divine_child", "weight": 0.85},
    {"primordial": "primordial:trickster", "weight": 0.70}
  ]
}
```
