# Aesop's Fables - Source Guide

## Overview
- **Origin**: Ancient Greece
- **Date**: ~6th century BCE
- **Type**: Fables
- **Author/Compiler**: Aesop

## Available Sources

### 1. George Fyler Townsend - STANDARD
**URL**: https://www.gutenberg.org/ebooks/11339
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: 284 fables

### 2. V.S. Vernon Jones - STANDARD
**URL**: https://www.gutenberg.org/ebooks/28
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: With illustrations

### 3. George Fyler Townsend - STANDARD
**URL**: https://sacred-texts.com/cla/aesop/
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: HTML version

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch aesops-fables
   ```

## File Naming Convention
- `aesops-fables-[translator]-[year].[ext]`
