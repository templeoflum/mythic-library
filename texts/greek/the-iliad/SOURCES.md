# The Iliad - Source Guide

## Overview
- **Origin**: Ancient Greece
- **Date**: c. 8th century BCE
- **Type**: Epic poetry
- **Author/Compiler**: Homer

## Available Sources

### 1. Samuel Butler - SCHOLARLY
**URL**: https://www.perseus.tufts.edu/hopper/text?doc=Perseus:text:1999.01.0134
- **Source**: Perseus Digital Library
- **Format**: HTML
- **Notes**: With Greek original available

### 2. Unknown - SCHOLARLY
**URL**: https://www.perseus.tufts.edu/hopper/text?doc=Perseus:text:1999.01.0133
- **Source**: Perseus Digital Library (Greek)
- **Format**: HTML
- **Notes**: Original Greek text

### 3. Samuel Butler - STANDARD
**URL**: https://www.gutenberg.org/ebooks/6130
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Prose translation

### 4. Alexander Pope (1715) - SCHOLARLY
**URL**: https://www.gutenberg.org/ebooks/2199
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Classic verse translation

### 5. Richmond Lattimore (1951) - SCHOLARLY
**URL**: https://archive.org/details/ilaborni00teletext
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Definitive modern verse translation

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch the-iliad
   ```

## File Naming Convention
- `the-iliad-[translator]-[year].[ext]`
