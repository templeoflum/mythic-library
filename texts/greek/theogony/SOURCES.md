# Theogony - Source Guide

## Overview
- **Origin**: Ancient Greece
- **Date**: c. 700 BCE
- **Type**: Creation myth
- **Author/Compiler**: Hesiod

## Available Sources

### 3. Hugh G. Evelyn-White - STANDARD

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch theogony
   ```

## File Naming Convention
- `theogony-[translator]-[year].[ext]`
