# The Odyssey - Source Guide

## Overview
- **Origin**: Ancient Greece
- **Date**: c. 8th century BCE
- **Type**: Epic poetry
- **Author/Compiler**: Homer

## Available Sources

### 1. Samuel Butler - SCHOLARLY
**URL**: https://www.perseus.tufts.edu/hopper/text?doc=Perseus:text:1999.01.0136
- **Source**: Perseus Digital Library
- **Format**: HTML
- **Notes**: With Greek original available

### 2. Unknown - SCHOLARLY
**URL**: https://www.perseus.tufts.edu/hopper/text?doc=Perseus:text:1999.01.0135
- **Source**: Perseus Digital Library (Greek)
- **Format**: HTML
- **Notes**: Original Greek text

### 3. Samuel Butler - STANDARD
**URL**: https://www.gutenberg.org/ebooks/1727
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Prose translation

### 4. Alexander Pope - SCHOLARLY
**URL**: https://www.gutenberg.org/ebooks/3160
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Classic verse translation

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch the-odyssey
   ```

## File Naming Convention
- `the-odyssey-[translator]-[year].[ext]`
