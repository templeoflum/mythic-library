# Tales of the North American Indians - Source Guide

## Overview
- **Origin**: North America (various tribes)
- **Date**: 1929
- **Type**: Native folklore
- **Author/Compiler**: Stith Thompson

## Available Sources

### 1. Unknown - SCHOLARLY
**URL**: https://archive.org/details/talesofnorthamer00thom
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Stith Thompson compilation, 1929

### 2. Unknown - STANDARD
**URL**: https://sacred-texts.com/nam/
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: Various Native American texts

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch tales-of-the-north-american-indians
   ```

## File Naming Convention
- `tales-of-the-north-american-indians-[translator]-[year].[ext]`
