# Kalevala - Source Guide

## Overview
- **Origin**: Finland/Karelia
- **Date**: 1835–1849
- **Type**: Epic poetry
- **Author/Compiler**: Elias Lönnrot

## Available Sources

### 1. John Martin Crawford (1888) - SCHOLARLY
**URL**: https://sacred-texts.com/neu/kveng/
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: Complete verse translation

### 2. John Martin Crawford - SCHOLARLY
**URL**: https://www.gutenberg.org/ebooks/5186
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Plain text version

### 3. W.F. Kirby - SCHOLARLY
**URL**: https://www.gutenberg.org/ebooks/7000
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Alternative translation

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch kalevala
   ```

## File Naming Convention
- `kalevala-[translator]-[year].[ext]`
