# Fēngshén Yǎnyì - Source Guide

## Overview
- **Origin**: China
- **Date**: c. 1550
- **Type**: Mythic novel
- **Author/Compiler**: Xu Zhonglin (attrib.)

## Available Sources

### 1. Gu Zhizhong - STANDARD
**URL**: https://archive.org/details/creationofgods0000unse
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Creation of the Gods - partial translation

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch fēngshén-yǎnyì
   ```

## File Naming Convention
- `fēngshén-yǎnyì-[translator]-[year].[ext]`
