# Journey to the West - Source Guide

## Overview
- **Origin**: China
- **Date**: 1592
- **Type**: Fantasy epic
- **Author/Compiler**: Wu Cheng'en

## Available Sources

### 1. W.J.F. Jenner - SCHOLARLY
**URL**: https://archive.org/details/journeytothewest0000wuch
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Complete translation

### 2. Timothy Richard (abridged) - STANDARD
**URL**: https://www.gutenberg.org/ebooks/60694
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Partial translation

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch journey-to-the-west
   ```

## File Naming Convention
- `journey-to-the-west-[translator]-[year].[ext]`
