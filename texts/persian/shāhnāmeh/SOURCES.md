# Shāhnāmeh - Source Guide

## Overview
- **Origin**: Persia (Iran)
- **Date**: 1010 CE
- **Type**: Epic poetry
- **Author/Compiler**: Ferdowsi

## Available Sources

### 1. Helen Zimmern (1883) - STANDARD
**URL**: https://archive.org/details/shahnamehtheepi00unkngoog
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Prose selections

### 2. Helen Zimmern - STANDARD
**URL**: https://sacred-texts.com/neu/shahnama.txt
- **Source**: Sacred Texts
- **Format**: TXT
- **Notes**: Same translation, plain text

### 3. Arthur George Warner & Edmond Warner (1905) - SCHOLARLY
**URL**: https://archive.org/details/shnmaoffirdaws01teletext
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Complete scholarly translation, 9 volumes

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch shāhnāmeh
   ```

## File Naming Convention
- `shāhnāmeh-[translator]-[year].[ext]`
