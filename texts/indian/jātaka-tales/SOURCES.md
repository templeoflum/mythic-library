# Jātaka Tales - Source Guide

## Overview
- **Origin**: India/South Asia
- **Date**: 3rd century BCE
- **Type**: Moral tales
- **Author/Compiler**: Anonymous

## Available Sources

### 1. E.B. Cowell et al. - SCHOLARLY
**URL**: https://sacred-texts.com/bud/j1/
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: Cambridge Jataka translation Vol 1

### 2. Ellen C. Babbitt - POPULAR
**URL**: https://www.gutenberg.org/ebooks/35665
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Adapted for children

### 3. E.B. Cowell - SCHOLARLY
**URL**: https://archive.org/details/jataka01cowegoog
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Cambridge translation, 6 volumes

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch jātaka-tales
   ```

## File Naming Convention
- `jātaka-tales-[translator]-[year].[ext]`
