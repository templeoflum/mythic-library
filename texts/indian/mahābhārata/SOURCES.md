# Mahābhārata - Source Guide

## Overview
- **Origin**: India
- **Date**: 400 BCE – 400 CE
- **Type**: Epic narrative
- **Author/Compiler**: Vyāsa

## Available Sources

### 1. Kisari Mohan Ganguli (1896) - SCHOLARLY
**URL**: https://sacred-texts.com/hin/maha/
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: Complete translation, 18 books

### 2. Kisari Mohan Ganguli - SCHOLARLY
**URL**: https://archive.org/details/mahabharata01teletext
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Multi-volume original publication

### 3. Kisari Mohan Ganguli (abridged) - STANDARD
**URL**: https://www.gutenberg.org/ebooks/15474
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Selections

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch mahābhārata
   ```

## File Naming Convention
- `mahābhārata-[translator]-[year].[ext]`
