# One Thousand and One Nights - Source Guide

## Overview
- **Origin**: Middle East & South Asia
- **Date**: 14th century
- **Type**: Folktale anthology
- **Author/Compiler**: Various

## Available Sources

### 1. Richard F. Burton (1885) - SCHOLARLY
**URL**: https://sacred-texts.com/neu/burt1k1/
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: Complete unexpurgated translation, 16 volumes

### 2. Richard F. Burton - SCHOLARLY
**URL**: https://www.gutenberg.org/ebooks/3435
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Volume 1

### 3. Andrew Lang - POPULAR
**URL**: https://www.gutenberg.org/ebooks/34206
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Adapted/abridged version

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch one-thousand-and-one-nights
   ```

## File Naming Convention
- `one-thousand-and-one-nights-[translator]-[year].[ext]`
