# Rāmāyaṇa - Source Guide

## Overview
- **Origin**: India
- **Date**: 300–100 BCE
- **Type**: Epic narrative
- **Author/Compiler**: Vālmīki

## Available Sources

### 1. Ralph T.H. Griffith - SCHOLARLY
**URL**: https://www.gutenberg.org/ebooks/24869
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Complete verse translation

### 2. Ralph T.H. Griffith (1870) - SCHOLARLY
**URL**: https://archive.org/details/ramayana01valm
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Original publication

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch rāmāyaṇa
   ```

## File Naming Convention
- `rāmāyaṇa-[translator]-[year].[ext]`
