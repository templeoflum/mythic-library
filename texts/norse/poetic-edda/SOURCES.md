# Poetic Edda - Source Guide

## Overview
- **Origin**: Iceland
- **Date**: 13th century
- **Type**: Mythic poems
- **Author/Compiler**: Anonymous

## Available Sources

### 1. Henry Adams Bellows (1936) - SCHOLARLY
**URL**: https://sacred-texts.com/neu/poe/
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: Standard scholarly translation

### 2. Henry Adams Bellows - SCHOLARLY
**URL**: https://www.gutenberg.org/ebooks/14726
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Plain text version

### 3. Olive Bray (1908) - SCHOLARLY
**URL**: https://archive.org/details/poeticedda00teletext
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Alternative translation

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch poetic-edda
   ```

## File Naming Convention
- `poetic-edda-[translator]-[year].[ext]`
