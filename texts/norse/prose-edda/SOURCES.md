# Prose Edda - Source Guide

## Overview
- **Origin**: Iceland
- **Date**: c. 1220 CE
- **Type**: Mythology manual
- **Author/Compiler**: Snorri Sturluson

## Available Sources

### 1. Arthur Gilchrist Brodeur (1916) - SCHOLARLY
**URL**: https://sacred-texts.com/neu/pre/
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: Definitive English translation

### 2. Arthur Gilchrist Brodeur - SCHOLARLY
**URL**: https://www.gutenberg.org/ebooks/18947
- **Source**: Project Gutenberg
- **Format**: TXT
- **Notes**: Plain text version

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch prose-edda
   ```

## File Naming Convention
- `prose-edda-[translator]-[year].[ext]`
