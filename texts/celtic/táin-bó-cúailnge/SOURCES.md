# Táin Bó Cúailnge - Source Guide

## Overview
- **Origin**: Ireland
- **Date**: 12th century (oral older)
- **Type**: Epic saga
- **Author/Compiler**: Anonymous

## Available Sources

### 1. Joseph Dunn (1914) - SCHOLARLY
**URL**: https://archive.org/details/tabornu00teletext
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Standard English translation

### 2. Various - STANDARD
**URL**: https://sacred-texts.com/neu/celt/cuch/
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: Cuchulain cycle selections

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch táin-bó-cúailnge
   ```

## File Naming Convention
- `táin-bó-cúailnge-[translator]-[year].[ext]`
