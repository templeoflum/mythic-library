# Popol Vuh - Source Guide

## Overview
- **Origin**: Maya (Guatemala)
- **Date**: 1550 (oral older)
- **Type**: Mythic cosmology
- **Author/Compiler**: Anonymous

## Available Sources

### 1. Lewis Spence - STANDARD
**URL**: https://sacred-texts.com/nam/pvuheng.htm
- **Source**: Sacred Texts
- **Format**: HTML
- **Notes**: Retelling

### 2. Delia Goetz & Sylvanus Morley (1950) - SCHOLARLY
**URL**: https://archive.org/details/popolvuh00teletext
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Based on Recinos translation

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch popol-vuh
   ```

## File Naming Convention
- `popol-vuh-[translator]-[year].[ext]`
