# Anansi Tales - Source Guide

## Overview
- **Origin**: West Africa / Caribbean
- **Date**: 19th c. collected
- **Type**: Trickster folklore
- **Author/Compiler**: Oral tradition

## Available Sources

### 1. Unknown - STANDARD

- **Notes**: Jamaican Anansi Stories

### 2. Unknown - STANDARD

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch anansi-tales
   ```

## File Naming Convention
- `anansi-tales-[translator]-[year].[ext]`
