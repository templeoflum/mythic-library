# Epic of Sundiata - Source Guide

## Overview
- **Origin**: Mali
- **Date**: 13th c. (written 20th c.)
- **Type**: Oral epic
- **Author/Compiler**: Oral tradition

## Available Sources

### 1. D.T. Niane/G.D. Pickett (1965) - SCHOLARLY
**URL**: https://archive.org/details/sundiataepicofol00nian
- **Source**: Internet Archive
- **Format**: PDF
- **Notes**: Definitive translation from Mandinka griot tradition

## Download Instructions

1. Visit each URL above
2. Download the text (PDF, TXT, or save HTML as text)
3. Save to this directory with descriptive filename
4. Run verification:
   ```
   python scripts/acquire.py fetch epic-of-sundiata
   ```

## File Naming Convention
- `epic-of-sundiata-[translator]-[year].[ext]`
